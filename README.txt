Harmonic Toolbox for Bilinear Systems

This toolbox provides quick access to harmonic analysis of bilinear systems. 
It includes core classes PhasorArray and PhasorSS for periodic modeling and simulation.

To set up your path, run installToolbox.m. 
After an update, it is recommended to reset the path with the same script. 
All other element of the path are preserved.

Then explore the demos in Exemples and the documentation in using "doc PhasorArray" or "doc "PhasorSS".
